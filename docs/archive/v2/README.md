# Cascadia V2 Archive

Cascadia v2 is closed. The final validation reached 95.744 mean base score and
did not meet the 100-point target.

This archive contains:

- `CASCADIA_V2_GOAL.txt`: the original v2 goal statement.
- `decisions/`: historical v2 decision records.
- `reports/`: generated v2 experiment evidence and validation reports.

Files inside the archive preserve their original contents, so some internal
paths still mention their pre-archive locations under `docs/v2`.

The maintained v2 handbooks remain in `docs/v2`. New strength work should use
the v3 plans in `docs/v3` and `cascadiav3`.
