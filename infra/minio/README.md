# MinIO object store

MinIO stores content-addressed inputs and execution-specific outputs on john1.
Credentials are generated into the runtime `secrets.env`, not this repository.

Pinned image:

`quay.io/minio/minio@sha256:14cea493d9a34af32f524e538b8346cf79f3321eff8e708c1e2960462bd8936e`

This is the final community server release published on the official Quay channel
(`RELEASE.2025-09-07T16-13-09Z`) and includes linux/arm64.
